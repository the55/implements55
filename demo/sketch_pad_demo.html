sketch pad jQuery plugin for Raphael

this is a jQuery plugin i wrote to regenerate a sketch on an interval you set. it will add a handy control so that you can pause it.

contain your sketch in a javascript function that takes a paper element. 

        var functionThatDraws = function(paper) {
          for(var i = 0; i < 500; i+=40) {
            for(var j = 0; j < 500; j+=40) {
              var width = Math.random()*20,
                height = Math.random()*20;
              paper.ellipse(i,j, width, height).
                attr('fill', 'red');
            }
          }
        };

pick your sketchpad element (only one-per-page, sorry). choose an interval and put it together. 

        $('#container').sketchPad55(functionThatDraws, 1000);

check the source of this page for an example.

check the source of raphael.sketchPad55.js for the options. ie: {noControl: true}
